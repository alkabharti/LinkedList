## Intersection Point in Y Shapped Linked Lists 

--------------------------------------------------------------------------------------------------------------------------------------------------------------


G Solution :

Time Complexity : 

```java
int intersectPoint(Node head1, Node head2)
{
       HashSet<Node> set = new HashSet<Node>();
       while(head1!=null)
       {
           set.add(head1);
           head1 = head1.next;
       }
       while(head2!=null)
       {
           if(set.contains(head2))
           {
               return head2.data;
           }
           head2 = head2.next;
       }
       return -1;
}
```
