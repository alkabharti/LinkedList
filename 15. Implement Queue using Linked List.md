## Implement Queue using Linked List : 

A Query Q is of 2 Types <br/>
(i) 1 x   (a query of this type means  pushing 'x' into the queue) <br/>
(ii) 2     (a query of this type means to pop an element from the queue and print the poped element)

**Example 1:**

Input: <br/>
Q = 5 <br/>
Queries = 1 2 1 3 2 1 4 2 <br/>
Output: 2 3 <br/>


**Example 2:**

Input: <br/>
Q = 4 <br/>
Queries = 1 2 2 2 1 3  <br/>
Output: 2 -1 <br/>

**Link :** https://practice.geeksforgeeks.org/problems/implement-queue-using-linked-list/1

Solution :

Time Complexity :

```java
class MyQueue
{
    QueueNode front, rear;
    
    //Function to push an element into the queue.
    void push(int a)
    {
          QueueNode node=new QueueNode(a);
          if(front==null)
              front=rear=node;
          else
          {
              rear.next=node;
              rear=node;
          }
    }
	
    //Function to pop front element from the queue.
    int pop()
    {
          if(front==null)
              return -1;
          int d=front.data;
          front=front.next;
          return d;
    }
}
```
